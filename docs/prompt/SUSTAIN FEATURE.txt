The ability to enter a key release action instead of a note in a given track position should be added.

In the note column, if no note is defined for a given step, this position defaults to containing "---". Similarly, if a key release action is specified in a given step, it should be displayed as "===". Entering a key release action in a given track position should be done by pressing the '-' (minus) key.

The key release action overrides the note value if one has been defined.

In the YAML format, this should be saved as a key "off" with a boolean value of true.

This must be accounted for in the "COPY TRACK", "PASTE TRACK", "SAVE SONG", and "LOAD SONG" functions.
